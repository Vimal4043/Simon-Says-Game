### **Simon Says Game**

A fun and interactive memory-based game created using HTML, CSS, and JavaScript.
Repeat the pattern shown by the game and try to reach the highest level!


### **Features**

* Clean and simple UI
* Increasing difficulty levels
* Highlight animations
* Fully responsive
* Made with pure JavaScript (no libraries)
